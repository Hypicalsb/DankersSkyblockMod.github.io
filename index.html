Dungeon Solvers

Slayer Trackers

Comprehensive tracking for all slayer types including drops, boss spawns, and profit calculations with detailed statistics.

🎣

Fishing Trackers

Track fishing drops, Jerry fishing events, fishing festivals, spooky fishing, and trophy fish with real-time statistics.

🔮

Mythological Tracker

Monitor mythological event progress, burrow detection, and Diana tracking with accurate spawn predictions.

Skill Trackers

Real-time skill experience tracking with hourly rates, progress bars, and time-to-level calculations for all skills.

Alerts & Notifications

📈

API Integration

Check player stats, bank balances, and trophy fish data using Hypixel API integration with detailed player profiles.

QOL Improvements

Dozens of quality of life features including reparty commands, ghost block fixes, and customizable GUI elements.

Powder Tracker

Track powder gains per hour, HotM tree progression, and mining statistics with detailed analysis tools.

⌨️ Commands & Keybinds

/dsm

Opens the main configuration menu to customize all mod settings and features.

/dhelp

Displays a comprehensive list of all available commands and their usage.

/loot [boss]

Shows loot tracking statistics for specific bosses or slayers.

/reparty

Automatically recreates your party after dungeon runs for seamless gameplay.

/hotmof [player]

Uses API to display total powder and HotM tree progression for any player.

/trophyfish [player]

Shows detailed trophy fish statistics and collection progress.

/bank [player]

Displays bank balance and purse information using API data.

/skilltracker [skill]

Toggle skill tracking and view experience rates for specific skills.

/lobbyskills

Shows skill levels without joining Skyblock (requires API key).

📥 Installation Guide

Install Forge 1.8.9

Download and install Minecraft Forge 1.8.9 from the official Forge website. This is required to run the mod.

Download DSM

Get the latest version of Danker's Skyblock Mod from our official GitHub releases page.

Place in Mods Folder

Put the downloaded .jar file in your .minecraft/mods folder. Create the folder if it doesn't exist.

Launch Minecraft

Start Minecraft with the Forge 1.8.9 profile and join Hypixel Skyblock to begin using DSM!
